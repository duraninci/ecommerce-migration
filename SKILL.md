## Quick Reference

### Platform Taxonomy

| Type | Examples | Best For | Migration Complexity |
|------|----------|----------|---------------------|
| **Open-Source** | Magento, WooCommerce | Custom, dev-heavy teams | HIGH |
| **Hosted SaaS** | Shopify Plus, BigCommerce | $500K-$20M+ brands | MEDIUM |
| **Enterprise** | VTEX, Oracle, SAP | Multi-national B2B | VERY HIGH |
| **Plug-and-Play** | Wix, Squarespace | Startups, low-SKU | LOW |

### Migration Drivers (The "Why")

1. **Performance bottlenecks** — Site can't handle traffic/catalog
2. **TCO escalation** — Plugin sprawl, hosting, dev retainers  
3. **Capability gaps** — "Our platform can't do that"

### Critical Success Factors

- **Data cleanliness** — 84% of failures trace to dirty data
- **301 redirect strategy** — Every legacy URL must map
- **Phased rollout** — Soft launch before full cutover
- **30-day monitoring** — First month is critical

## Key Frameworks

### Migration Readiness Checklist

```markdown
□ Data cleanliness audit (duplicates, variants, formats)
□ Custom functionality inventory
□ Integration dependencies mapped (ERP, PIM, 3PL, CRM)
□ SEO baseline documented (rankings, traffic, URLs)
□ Team roles assigned (PM, Data Architect, SEO Lead)
□ Success KPIs defined
□ Rollback plan documented
```

### TCO Calculation Framework

```
Total Cost = Direct + Indirect + Operational

Direct:
- Platform licensing
- Hosting fees
- SSL certificates

Indirect:
- Third-party apps
- Payment gateway fees
- Middleware licensing

Operational:
- Developer retainers
- Internal staff hours
- Downtime revenue loss
```

### 301 Redirect Template

```
Old URL → New URL → Redirect Type
/products/old-slug → /products/new-slug → 301
/category/old-cat → /collections/new-cat → 301
/pages/about-us → /pages/about → 301
```
