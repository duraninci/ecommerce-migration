## 💡 Real-World Use Cases

### Scenario 1: Magento → Shopify Plus Migration
**Situation:** $3M/year brand on Magento 1 (EOL), 50K SKUs, complex product configurator

**This skill provides:**
1. Complexity assessment (HIGH — custom functionality dependency)
2. Data migration hierarchy (products → customers → orders)
3. Custom functionality alternatives on Shopify
4. SEO preservation strategy (301 mapping for 50K products)
5. Phased rollout recommendation (soft launch → full cutover)

### Scenario 2: SEO Traffic Preservation
**Situation:** Ecommerce site ranking #1-3 for 200+ commercial keywords, terrified of losing rankings

**This skill provides:**
1. Pre-migration SEO baseline documentation
2. URL mapping templates for all content types
3. 301 redirect implementation checklist
4. Crawl budget optimization for new platform
5. 90-day monitoring protocol with recovery triggers

### Scenario 3: B2B Industrial Distributor
**Situation:** $50M wholesaler needs quotes, corporate accounts, custom pricing by customer

**This skill provides:**
1. B2B platform comparison (BigCommerce B2B Edition vs. Shopify B2B)
2. Quote-to-cash workflow mapping
3. ERP integration architecture
4. Customer tier implementation
5. Approval workflow design

### With Claude (claude.ai)

1. Download the skill files
2. Add `SKILL.md` to your Claude Project's knowledge base
3. Add relevant reference files from `references/` folder
4. Start asking about migrations

### With Claude Code / OpenClaw

```bash

git clone https://github.com/duraninci/ecommerce-migration.git ~/.openclaw/skills/ecommerce-migration

# Or copy to existing skills folder
cp -r ecommerce-migration ~/.openclaw/skills/
```

Skills auto-load, or reference directly:
```
Review this migration plan using ~/.openclaw/skills/ecommerce-migration/SKILL.md
```

